# CalorieTracker

A modern web application for tracking daily calorie intake and nutritional information. Built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Track daily calorie intake
- AI-powered food analysis using Google's Gemini API
- Beautiful, responsive UI with dark mode support
- Real-time nutrition tracking
- Daily and weekly goal setting
- Meal categorization (breakfast, lunch, dinner, snacks)

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Shadcn/ui Components
- Google Gemini API
- React Context for state management

## Getting Started

1. Clone the repository:
```bash
git clone [your-repo-url]
```

2. Install dependencies:
```bash
npm install
# or
pnpm install
```

3. Set up environment variables:
Create a `.env.local` file in the root directory and add:
```
GEMINI_API_KEY=your_gemini_api_key
```

4. Run the development server:
```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
